# PoE2 做装模拟器原型

静态网页入口：

- `index.html`

当前版本实现了逐步做装状态机：

- 普通、魔法、稀有的前缀/后缀槽位限制
- 词缀按底材、物品等级、词缀类型、词缀组、权重过滤
- 蜕变、增幅、点金、富豪、崇高、混沌、剥离、神圣
- 机会石、瓦尔宝珠、破溃石、巧匠石、卡兰德的魔镜
- 护甲片、磨刀石、奥术蚀刻石等品质通货
- 普通/高级/完美精髓
- 巧匠插槽和样例符文镶嵌
- 左旋/右旋崇高、富豪、剥离、消抹预兆
- 消减预兆：下一颗混沌石移除最低等级词缀
- 左旋/右旋炼金、同质化、强效崇高、强效剥离、远古、机遇、结晶预兆
- 左旋/右旋亵渎预兆和光明预兆
- 深渊回响预兆、腐烂预兆
- 锁骨、肋骨、颚骨三类亵渎材料，含远古/保存完好/啃噬限制
- 高级/完美通货的最低词缀等级过滤
- 每一步操作历史、待触发预兆、亵渎词缀、品质、插槽、腐化、镜像、破碎状态和当前可用词缀池
- 高级/完美通货按当前可用词缀池和权重抽取，不再在样例数据里假性固定 T1
- 右侧按钮会随通货等阶显示为高级/完美通货名，避免误把完美崇高当普通崇高

数据说明：

- 目前内置戒指、鞋子、项链、腰带、弓、法杖和一批样例词缀。
- 数据结构按编年史常见字段建模：底材、物品等级、前缀/后缀、词缀组、权重、等级、标签、数值范围。
- 后续接全量编年史数据时，优先替换 `crafting-core.js` 里的 `BASES`、`MODIFIERS` 和 `DESECRATED_MODIFIERS` 数据表，通货状态机可以继续复用。
- 机会石和瓦尔宝珠的随机分支已经接入状态机，但具体分支权重需要后续用真实游戏数据校准。
